This is probably the only QWC2 Docker git repository you will find on creating a QWC2 Docker and run it on a Cloud instance.  
This is a responsive website(built with ReactJS and OpenLayers) with QWC2 Docker, with map projects served from QGIS server, and finally deployed on a docker container on a Linux OS on Microsoft Azure instance and secured with Cloudflare DNS.  
Tools used:
-AWS EC2 or Azure  
-Docker  
-ReactJS  
-Cloudfare DNS setup

QGIS is a free and open-source Geographic Information System (GIS) software designed for creating, editing, storing, analyzing, and visualizing geospatial data. The software supports numerous raster and vector formats, as well as point clouds, and is utilized in urban planning, environmental conservation, and transportation.
